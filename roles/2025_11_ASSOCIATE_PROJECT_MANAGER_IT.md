# UL Solutions Associate Project Manager Apprentice (IT)

**Objective:** This document outlines a structured learning plan to build the necessary skills and knowledge for the Associate Project Manager Apprentice (IT) role at UL Solutions. The plan is phased to align with key milestones: the meet-and-greet with the hiring manager and the start date.

**Key Context:** 
- UL Solutions is a Microsoft-first organization, using the Power Platform (Power BI, Power Automate, Power Apps), Azure DevOps, ServiceNow, and MS Project as core tools
- The role supports IT Vendor Management and Optimization in establishing new frameworks and processes—opportunity to contribute to foundational work

### Key Vocabulary & Concepts to Master

#### **Power Platform Terms:**
- **Power BI:** Business intelligence and data visualization platform
- **Power Apps:** Low-code platform for building custom business apps
- **Power Automate:** Workflow automation tool (formerly Microsoft Flow)
- **Canvas App:** Freeform app design starting with blank canvas
- **Model-Driven App:** App built on top of Dataverse with structured data model
- **Cloud Flow:** Automated workflow triggered by events
- **Dataverse:** Microsoft's cloud database for Power Platform
- **DAX:** Data Analysis Expressions—formula language for Power BI
- **Power Query:** Data transformation tool in Power BI

#### **Vendor Management Terms:**
- **RFP:** Request for Proposal—document soliciting vendor bids
- **RFI:** Request for Information—preliminary document to learn about vendor capabilities
- **SLA:** Service Level Agreement—contract defining service standards
- **SOW:** Statement of Work—detailed description of services vendor will provide
- **TCO:** Total Cost of Ownership—full cost including hidden costs
- **Vendor Risk:** Potential issues (financial instability, security vulnerabilities, single-source dependency)
- **Contract Lifecycle:** Identification → Selection → Negotiation → Execution → Management → Renewal/Termination
- **Vendor Scorecard:** Tool for evaluating vendor performance against KPIs
- **Spend Analysis:** Review of spending patterns to identify savings opportunities
- **Category Management:** Organizing spend into categories for strategic sourcing

#### **IT Service Management Terms:**
- **ITSM:** IT Service Management—processes for delivering IT services
- **Incident:** Unplanned interruption or reduction in service quality
- **Change Management:** Process for controlling changes to IT environment
- **Service Catalog:** Menu of IT services available to users
- **CMDB:** Configuration Management Database—repository of IT assets and relationships
